What is Suspicious?

Suspicious

Suspicious is a web application that allows users to submit and analyze various types of data, such as emails, files, IP addresses, and URLs.

How does it work ?

Suspicious uses Cortex an open source application that launches analysis on various objects.

When a Reporter sends a mail to the Suspicious mail address ({% suspicious_email %}) the mail is splitted in multiple parts and each part is analyzed

For example the mail body is analyzed by the Yara Rules that searches for any malicious content that could define the mail as Phishing or Spam

Then all the mail parts are sent to the Cortex Analyzers that will analyze the mail and return a report for each

Finally the Suspicious app will calculate a score from the reports of cortex and make a conclusion on the overall

Note that it works exactly the same for files, IP addresses, Hashes and URLs

Classification

Suspicious

  • Dangerous
  • Suspicious
  • Inconclusive
  • Safe

The classification is made by the score of the reports

For example if a mail is classified as Dangerous it means that the mail has a high score of malicious content

On the other hand if a mail is classified as Safe it means that the mail has a low score of malicious content

Mails classed as Inconclusive are mails that have a score of malicious content that is neither high nor low and also it can be that their sources are not known

Finally mails classed as Suspicious are mails that have a score of malicious content that is neither high nor low but their sources are known and that it tends to be dangerous


Dangerosity

Each classification has a dangerosity

Dangerous means that the Mail / File can not be opened and that anything in it can not be trusted

Suspicious means that the Mail / File should not be opened and that anything in it should not be trusted

Inconclusive means that the Mail / File can be opened but that anything in it should not be trusted

Safe means that the Mail / File can be opened and that anything in it can be trusted

The classification and dangerosity are the same also for Ip Addresses and Urls

What is phishing ?

A way of getting your informations

Phishing is a type of social engineering attack used to steal user data, such as login credentials and credit card numbers. The attacker pretends to be a trusted entity and tricks the victim into opening an email, instant message, or text message. The victim is then deceived into clicking a malicious link, which can result in malware installation, ransomware attacks, or the disclosure of sensitive information.

Phishing attacks have become increasingly sophisticated and difficult to identify

That is the reason to use Suspicious and to have help on how to categorize a mail you receive

File analysis ?

How are file analyzed

When a file is sent to Suspicious, we calculate its Hash (A unique identifier for a file) that we compare and search for in various global databases to retrieve informations if it was already submitted elsewhere

It also is compared to our own database to see it it was already submitted by a Reporter

The file is never opened or executed, it is only analyzed by its Hash and Metadatas (informations on the file to see its type, date of creation, last moodification date, ect)
